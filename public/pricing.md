# Pricing — Namespace

## Offchain Subnames
- **Price**: $0 (always free)
- **Limits**: Unlimited subnames
- **Features**: Gasless issuance, CCIP-Read resolution, quick setup, simple management
- **Best for**: Wallets, apps, and communities that need to onboard users fast

## Onchain Subnames
- **Price**: Free if minting price is set to zero; otherwise gas fees apply
- **Limits**: Unlimited subnames
- **Features**: Minted as NFTs on Ethereum, Base, or Optimism; self-custody; transferable
- **ENS DAO fee**: If a partner charges for subname minting, Namespace takes a 5% fee on that revenue
- **Best for**: Premium names, collectibles, and tradable identities

## SDK / API
- **Price**: $0 (free to use)
- **Limits**: No rate limits or usage caps
- **Features**: JavaScript/TypeScript SDK and REST API for programmatic subname registration and management
- **Best for**: Developers integrating ENS naming into wallets, rollups, and apps

## Custom Solutions
- **Price**: Custom — contact sales
- **Features**: Tailored ENS naming integrations for wallets, L2 chains, and apps
- **ENS DAO subsidy**: Partially subsidized by ENS DAO for qualifying projects
- **Best for**: Enterprises and ecosystems needing bespoke identity infrastructure
